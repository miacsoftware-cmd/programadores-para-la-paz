El login permite identificar a un usuario antes de darle acceso a las funciones de una aplicación. De esta manera solo las personas autorizadas pueden realizar determinadas acciones. Algunas rutas deben estar protegidas porque contienen información privada o permiten modificar datos del sistema. Esto ayuda a evitar accesos no autorizados y protege la integridad de la plataforma. No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de la comunidad.
