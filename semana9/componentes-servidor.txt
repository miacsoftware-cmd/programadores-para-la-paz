Sistema operativo:
Es el software principal que administra los recursos del computador o servidor. Permite ejecutar programas y controlar el funcionamiento del hardware.

Node.js:
Es un entorno de ejecución que permite correr código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y respuestas que recibirán los usuarios.

Puerto:
Es un canal de comunicación que utiliza el servidor para recibir solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros de eventos y actividades del servidor. Ayudan a monitorear el funcionamiento y detectar errores.
