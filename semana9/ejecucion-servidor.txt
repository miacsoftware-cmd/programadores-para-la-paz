Al ejecutar el comando node server.js, se inicia el servidor creado con Express.

En la terminal aparece el mensaje:
"Servidor ejecutándose en puerto 3000".

Cuando visito la ruta http://localhost:3000/estado, el servidor responde con un objeto JSON que indica que el servidor está funcionando correctamente.

Respuesta obtenida:

{
  "estado": "Servidor funcionando",
  "servicio": "API comunitaria"
}
